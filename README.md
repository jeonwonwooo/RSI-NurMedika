## 📌 Deskripsi Karya

Website **RSI Nur Medika** merupakan sistem informasi rumah sakit berbasis web yang dirancang untuk mempermudah proses administrasi dan pelayanan kesehatan secara digital.

Sistem ini membantu dalam pengelolaan data pasien, pendaftaran online, jadwal dokter, rekam medis, serta pengolahan administrasi secara terstruktur dan efisien. Dengan adanya website ini, proses pelayanan menjadi lebih cepat, akurat, dan meminimalisir kesalahan pencatatan manual.

### ✨ Fitur Utama:
- Pendaftaran pasien secara online  
- Manajemen data pasien  
- Informasi jadwal dokter  
- Pengelolaan rekam medis  
- Sistem administrasi dan laporan  

Website ini dikembangkan untuk mendukung transformasi digital di lingkungan rumah sakit agar pelayanan menjadi lebih modern, efektif, dan mudah diakses oleh masyarakat.
